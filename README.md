## 分库中的水平拆分 ( 读写分离 )
test02-sharding-jpa

```text
1.数据库原理
    主库: 开启binlog ( sql操作的二进制日志 )
    从库: 监听主库的二进制日志变化，当变化时在从库执行对应的sql
2.应用程序原理
    主库数据源
        写
        开启事务时，读写都在主库操作
    从库数据源
        只读
```
