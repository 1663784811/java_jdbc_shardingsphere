## 读写分离
读写分离示例 test01

一个主库
两个从库

数据库原理:
```text
主库: 开启binlog ( sql操作的二进制日志 )

从库: 监听主库的二进制日志变化，当变化时在从库执行对应的sql

```
应用程序原理:
```text
主库数据源
    写
    开启事务时，读写都在主库操作

从库数据源
    只读

```
